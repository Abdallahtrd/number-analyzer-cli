# OVERVIEW
## Number Analyzer (Python CLI)

A simple command-line Python program that collects numbers from the user and performs basic analysis.

## Features
	•	Add numbers to a list
	•	Calculate the average of entered numbers
	•	Handles invalid input using try/except
	•	Prevents errors when calculating the average of an empty list

## Concepts Practiced
	•	while loops
	•	lists
	•	functions
	•	parameters and arguments
	•	error handling
	•	conditional statements
